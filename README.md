Sistema de Autenticación Full-Stack

Descripción

Este proyecto es una aplicación web donde los usuarios pueden registrarse mediante un sistema completo que conecta frontend y backend.

El objetivo fue entender cómo funciona una aplicación real, desde la interfaz hasta la base de datos.

---

Tecnologías utilizadas

* React (Frontend)
* Django (Backend)
* Django REST Framework (API)
* SQLite (Base de datos)

---

Funcionalidades

* Registro de usuarios
* Validación de datos
* Conexión entre frontend y backend mediante API REST
* Manejo de errores básicos

---

Lo que aprendí

* Cómo conectar React con Django
* Cómo crear y consumir APIs
* Manejo de base de datos
* Resolución de errores reales (CORS, CSRF)



 Cómo ejecutar el proyecto

Backend:

cd backend
python manage.py runserver


Frontend:

cd frontend
npm install
npm run dev


Estado del proyecto

Proyecto funcional en desarrollo, enfocado en aprendizaje y mejora continua.
